# CoopTest

This project comprises automated tests to verify the functionality of the loan calculator on the Coop pank website.

## Getting Started

To run the tests, follow these steps:

1. Ensure that you have the required dependencies installed: Selenium WebDriver, ChromeDriver, JUnit.
2. Clone the repository: `git clone https://github.com/your_project.git`
3. Open the project in  (IDE).
4. To run tests : execute mvn test

## Test Scenarios

### 1. Test for checking the display of maximum loan amount
### 2. Test for checking the message in case of low income
### 3. Test for checking the input of negative numbers
### 4. Test for checking the message in case of to high income
